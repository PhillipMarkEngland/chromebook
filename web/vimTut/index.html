How to use Vim

Command Mode and Insert Mode

When you open Vim you will be in command mode

To be able to type you will need to go into insert mode

Press "i" to move into insert move

Press "esc" to go back into command mode

You can press "o" to enter insert mode on a newline below your cursor, or "O" to enter insert mode on a newline above your cursor.

How to Exit Vim and Save your File

In command mode, press ":" to enter a command

If you have not made any changes to your file, press ":q" to exit Vim

If you have made changes to your file you can type ":wq" to exit and save your changes.

If you want to exit without saving your changes, just type ":q!"

If you want to save your changes without exiting vim, then type ":w" to write your changes to the file.

Moving Around in Vim

You can use the "h" "j" "k" and "l" keys to navigate through your file

Press "gg" in command mode to go to the top of the file and press "G" to go to the bottom of the file.

You can use "{" and "}" to skip blocks of code. "{" to go backwards in your file and "}" to go forwards in your file.

You can place numbers in front of commands to repeat the command a given number of times. For example, you can do "h20" to move to the left 20 times. You can do this with most commands in vim.

You can press "w" to move to the next word in a line.

You can press "b" to go back a word.

You can press "0" to go to the start of a line or "$" to move to the end of a line.

Deleting and Undoing Changes

In command mode, press "dd" to delete a line. This will also copy the deleted line to your clipboard.

You can press "u" to undo changes made to a file.

You can press "ctrl+r" to redo a change made to a file.

You and press "yy" to copy a given line.

You can then press "p" to paste that line below your cursor or "P" to paste the line above your cursor.

Entering Visual Mode

Press "V" to enter visual mode.

This allows you to select multiple lines of code at onde and visually see which lines of code you are selecting.
